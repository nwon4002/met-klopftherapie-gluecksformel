## Was ist die MET-Klopftherapie eigentlich?

MET kombiniert zwei Welten: die westliche Psychotherapie (konkret Gestalttherapie) und die chinesische Akupressur. Die Grundannahme: Negative Gefühle wie Angst, Trauer oder Wut hinterlassen Energieblockaden im Körper. Indem man bestimmte Meridianpunkte beklopft – während man das Gefühl bewusst festhält – löst sich die Blockade auf.

Was klingt wie ein esoterisches Konzept, hat praktisch eine klare Struktur:

1. Man benennt das Gefühl, das einen belastet.
2. Man schätzt seine Intensität auf einer Skala von 0 bis 10.
3. Man klopft mit den Fingerspitzen sechs spezifische Punkte am Oberkörper und Gesicht.
4. Man überprüft nach dem Klopfen, ob die Intensität gesunken ist.
5. Man wiederholt den Vorgang, bis das Gefühl deutlich nachlässt oder verschwindet.

## Wie geht das Klopfen konkret? Ein Einstieg

Du musst kein Vorwissen mitbringen. Rainer Franke erklärt im Kurs alles Schritt für Schritt. Zum Eingewöhnen hier der grundlegende Ablauf:

1. **Gefühl benennen**: Welches Gefühl drückt gerade? Angst? Wut? Trauer? Schuldgefühle?
2. **Intensität einschätzen**: Wie stark ist es auf einer Skala von 0 bis 10?
3. **Klopfsatz formulieren**: Ein kurzer Satz, der das Gefühl beschreibt – z.B. „diese Angst vor dem Gespräch"
4. **Klopfpunkte stimulieren**: Mit Zeige- und Mittelfinger sanft auf die sechs Punkte klopfen, während du den Klopfsatz innerlich festhältst oder murmelt
5. **Intensität prüfen**: Hat die Zahl sich verändert? Gut. Dann weiter klopfen, bis sie deutlich sinkt.

Was das Klopfen von reinen Affirmationen unterscheidet: Du musst das Gefühl nicht „wegdenken", du klopfst es buchstäblich weg – oder lässt es zumindest kleiner werden.
